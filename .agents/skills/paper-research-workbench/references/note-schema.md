# Note Schema

`note.md` should separate evidence from interpretation.

## Required Sections

- `Summary`: problem, method, result, limitation.
- `Evidence Notes / Observed`: direct facts from paper text.
- `Evidence Notes / Inferred`: reasoned interpretation from facts.
- `Evidence Notes / Open Questions`: missing details or uncertainty.
- `Reproduction Hooks`: data, metrics, baseline, implementation clues.
- `Follow-up Ideas`: concrete extension opportunities.

## Writing Rules

- Cite page sections when possible from `_artifacts/sections.json`.
- Keep statements falsifiable and testable.
- If confidence is low, mark it in `Open Questions` instead of asserting.
