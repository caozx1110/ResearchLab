# Idea Rubric

Use this rubric when filling `ideas.md`.

Score each dimension 1-5:

- `Novelty`: how original compared with obvious baselines.
- `Leverage`: how much existing infra/data can be reused.
- `Evaluation clarity`: how clearly success can be measured.
- `Engineering complexity`: implementation burden (higher is harder).
- `Data burden`: new data collection burden (higher is harder).

## Ranking Heuristic

Compute a rough score:

`priority = novelty + leverage + evaluation_clarity - engineering_complexity - data_burden`

Use as ranking guidance only, not as absolute truth.
